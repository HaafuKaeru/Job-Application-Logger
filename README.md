# Job Application Logger

A small terminal based program where the user inputs a job posting link (currently working only for Linkedin pages) and relevant data such as job position name, company name, location, etc. 
are automatically fed into a personal Google sheet specified in the backend. No more manual entries 😌 (at least until I use up all the free Sheety API requests...)

Web scraping with <a href="https://beautiful-soup-4.readthedocs.io/en/latest/">BeautifulSoup</a>

Google sheet access with <a href="https://sheety.co/">Sheety API</a>
